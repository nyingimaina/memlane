# Track Specification: Web Dashboard Implementation

## Overview
This track focuses on building the Memlane web interface using Next.js, TypeScript, and the `jattac.libs.web.zest-*` component library. The dashboard will provide a professional, corporate interface for managing backup jobs and viewing real-time status updates from the C# backend.

## Objectives
- Scaffold a Next.js 14+ (App Router) project with TypeScript.
- Integrate the `jattac.libs.web.zest-*` suite for a professional UI.
- Implement the Logic-Repository pattern for frontend state management.
- Establish a real-time connection to the backend SignalR `JobHub`.
- Display a responsive job monitoring dashboard.

## Key Components
- **Framework:** Next.js (TypeScript).
- **UI Suite:** Zest (Textbox, Button, ResponsiveLayout, SidekickMenu, ResponsiveTable, OverflowMenu).
- **State Management:** Logic-Repository pattern.
- **Communication:** Axios for REST API and `@microsoft/signalr` for real-time updates.

## Tech Stack Alignment
- Next.js (TypeScript)
- Vanilla CSS
- SignalR Client
- Zest Component Library
